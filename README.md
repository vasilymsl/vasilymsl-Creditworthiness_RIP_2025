# Разработка интернет-приложений (РИП) 2025

## Оценка кредитоспособности заемщика

Разработанное приложение позволит лечащим врачам создавать и формировать заявки на расчеты кредитоспособности для пациентов, а главным врачам завершать или отклонять данные заявки.
